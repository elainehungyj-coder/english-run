# English Run

Public practice tool for meeting-script rehearsal.

This repository should only contain the tool itself. Do not upload `Cynthia_private_script_data.json` to GitHub.

Use on iPhone:

1. Open the GitHub Pages URL in Safari.
2. Add the site to Home Screen.
3. Open the app and import `Cynthia_private_script_data.json`.
4. The imported script, progress, and vocabulary stay in local storage on that device.
